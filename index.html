CFD의 과정과 구성요소

전산 유체 역학(CFD, Computational Fluid Dynamics)은 컴퓨터를 사용하여 유체의 운동, 열 전달, 화학 반응 등 다양한 물리적 현상을 수치적으로 해석하고 예측하는 학문이자 강력한 엔지니어링 도구입니다. CFD는 항공우주, 자동차, 에너지, 화학공업, 바이오프로세스 등 광범위한 분야에서 제품 설계, 성능 최적화, 문제 해결에 핵심적인 역할을 수행합니다. [Ansys]

본 연구실에서는 특히 바이오리액터 설계 및 최적화, 약물 전달 시스템 개발 등 생물공학(Bioprocess Engineering) 분야의 난제 해결을 위해 CFD를 적극적으로 활용하고 있습니다. (Bioprocess CFD 소개 보기)

CFD 해석의 주요 과정

CFD 해석은 일반적으로 다음과 같은 세 가지 주요 단계로 구성됩니다. 각 단계는 상호 유기적으로 연결되어 있으며, 정확하고 신뢰성 있는 결과를 얻기 위해서는 각 단계에 대한 깊이 있는 이해와 세심한 주의가 필요합니다. [1, 2, 3]

1. 전처리 (Pre-processing)

해석을 위한 준비 단계로, 실제 물리 문제를 컴퓨터가 이해하고 계산할 수 있는 형태로 변환하는 과정입니다. 이 단계의 품질이 전체 CFD 해석 결과의 정확도와 신뢰성에 결정적인 영향을 미칩니다. [1, 5]

  • 형상 정의 (Geometry Definition): 분석 대상 시스템의 2D 또는 3D 기하학적 형상을 생성하거나, 외부 CAD(Computer-Aided Design) 데이터를 불러와 정리합니다. 유동 해석에 불필요한 미세 형상은 단순화(de-featuring)하여 계산 효율을 높입니다. [1]
  • 격자 생성 (Mesh Generation): 정의된 계산 영역(Domain)을 유한한 크기의 작은 제어 체적(셀 또는 요소)으로 분할합니다. 격자의 종류(정형/비정형), 밀도, 품질(왜곡도, 종횡비 등)은 해석 결과의 정확도와 계산 시간에 직접적인 영향을 미칩니다. [1, 6]
  • 물리 모델 설정 (Physics Modeling): 해석하고자 하는 유동의 특성(예: 정상/비정상 상태, 층류/난류 유동), 열 전달 유무, 상 변화, 화학 반응 등을 고려하여 적절한 수학적 모델(예: 난류 모델, 다상 유동 모델, 에너지 방정식)을 선택하고 관련 파라미터를 설정합니다. [1, 4, 9]
  • 경계 조건 설정 (Boundary Conditions): 계산 영역의 경계면(입구, 출구, 벽면, 대칭면 등)에서 유동 변수(속도, 압력, 온도, 농도 등)의 값이나 물리적 거동을 정의합니다. 이는 미분방정식의 해를 유일하게 결정하는 데 필수적입니다. [1, 5]
  • 초기 조건 설정 (Initial Conditions): 비정상 상태 해석의 경우, 계산 시작 시점(t=0)에서의 유동장 전체 상태를 정의합니다. 정상 상태 해석에서도 수렴을 돕기 위해 적절한 초기값을 설정합니다. [1]

2. 해석 (Solving)

전처리 단계에서 설정된 모델과 조건에 따라, 유체의 거동을 기술하는 지배 방정식(질량, 운동량, 에너지 보존 방정식 등)을 수치적으로 풀어 각 격자점 또는 셀에서의 유동 변수(속도, 압력, 온도 등) 값을 계산하는 단계입니다. [1, 4]

  • 지배 방정식 (Governing Equations): 유체 유동을 지배하는 기본적인 물리 법칙(질량 보존, 운동량 보존 - 나비에-스토크스 방정식, 에너지 보존)을 수학적으로 표현한 편미분 방정식입니다. [NASA, CFD Online]
  • 이산화 (Discretization): 연속적인 미분 형태의 지배 방정식을 컴퓨터가 처리할 수 있는 대수 방정식 형태로 변환하는 과정입니다. 주로 유한 체적법(FVM), 유한 요소법(FEM), 유한 차분법(FDM) 등이 사용되며, FVM이 CFD 분야에서 널리 쓰입니다. [1, 6, 7]
  • 수치 해법 (Numerical Solution Algorithm): 이산화된 대수 방정식 시스템을 풀기 위한 알고리즘입니다. 비압축성 유동에서는 압력-속도 연계 알고리즘(SIMPLE, PISO 등)이 사용되며, 반복 계산을 통해 해를 구합니다. [1, 4]
  • 수렴 판정 (Convergence Criteria): 반복 계산 과정에서 해가 더 이상 변하지 않고 안정된 상태에 도달했는지 (수렴했는지) 판단하는 기준입니다. 주로 각 방정식의 잔차(Residual)가 특정 값 이하로 감소하거나, 주요 물리량의 변화가 없을 때 수렴으로 간주합니다. [4]

3. 후처리 (Post-processing)

솔버에서 계산된 방대한 양의 수치 데이터를 분석하고 시각화하여 유동 특성을 파악하고, 공학적 통찰력을 얻으며, 설계 검증 및 개선을 위한 결정을 내리는 최종 단계입니다. [1, 2, 3]

  • 결과 시각화 (Results Visualization): 속도 벡터장, 압력 등고선, 유선(streamlines), 온도 분포, 와류 구조 등을 2D 또는 3D 그래픽으로 표현하여 유동 현상을 직관적으로 이해합니다. 애니메이션을 통해 비정상 유동의 시간적 변화도 관찰할 수 있습니다. [1, 3]
  • 정량적 데이터 추출 (Quantitative Data Extraction): 특정 지점, 선, 면, 또는 체적에서의 물리량 값(예: 특정 지점의 속도, 면을 통과하는 유량), 평균값, 최대/최소값, 힘(항력, 양력), 모멘트, 열 전달 계수 등 필요한 공학적 데이터를 추출합니다. [1, 2]
  • 검증 및 타당성 조사 (Verification & Validation - V&V): 시뮬레이션 결과의 신뢰성을 확보하는 매우 중요한 과정입니다. 검증(Verification)은 수치 해법이 수학적 모델을 올바르게 풀었는지(예: 격자 수렴성 테스트), 확인(Validation)은 시뮬레이션 모델이 실제 물리 현상을 정확하게 예측하는지(예: 실험 데이터와의 비교)를 평가합니다. [1, 2, 4]
