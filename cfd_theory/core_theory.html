핵심 이론

CFD 해석의 근간은 유체의 물리적 거동을 기술하는 지배 방정식, 이 연속적인 방정식을 컴퓨터 계산이 가능한 이산적인 형태로 변환하는 이산화 기법, 그리고 복잡한 유체 현상, 특히 난류를 효과적으로 모사하기 위한 모델에 있습니다.

3.1 지배 방정식 (Governing Equations)

CFD 해석의 출발점은 유체의 질량, 운동량, 에너지 보존 법칙을 나타내는 비선형 편미분 방정식입니다.

연속 방정식 (Continuity Equation)

질량 보존 법칙을 나타냅니다. 특정 체적 내 유체의 질량 변화율은 그 체적 경계를 통해 유입/유출되는 질량 유량의 순 변화와 같다는 원리입니다.

일반 형태: ∂ρ/∂t + ∇ · (ρu) = 0

비압축성 유동(ρ = 일정)의 경우: ∇ · u = 0

운동량 방정식 (Momentum Equation) / 나비에-스토크스 방정식 (Navier-Stokes Equations)

뉴턴의 제2법칙(F = ma)을 유체 요소에 적용한 것입니다. 점성 유체에 대한 운동량 방정식이 나비에-스토크스 방정식입니다.

나비에-스토크스 방정식 (비압축성 뉴턴 유체):
ρ(∂u/∂t + (u · ∇)u) = -∇p + µ∇²u + ρfb

오일러 방정식 (비점성 유체):
ρ(∂u/∂t + (u · ∇)u) = -∇p + ρfb

에너지 방정식 (Energy Equation)

열역학 제1법칙을 유체 요소에 적용한 것입니다. 유동장의 온도 분포 및 열전달 해석에 필수적입니다.

일반 형태 (전체 엔탈피 htotal):
∂(ρhtot)/∂t + ∇ · (ρuhtot) = ∂p/∂t + ∇ · (λ∇T) + τ:∇u + SE

3.2 이산화 기법 (Discretization Methods)

지배 방정식의 연속적인 형태를 컴퓨터 계산이 가능한 대수 방정식 시스템으로 변환하는 과정입니다. 'CFD 프로세스 개요'의 '솔버' 섹션에서 주요 기법(FVM, FEM, FDM)에 대한 자세한 설명을 참고하십시오. (또는 여기를 클릭)

3.3 난류 모델 (Turbulence Modeling)

실제 공학 유동의 대부분은 불규칙하고 복잡한 난류입니다. 난류 모델은 이러한 난류의 영향을 효율적으로 모사하기 위해 사용됩니다.

RANS (Reynolds-Averaged Navier-Stokes) 모델

시간 평균된 나비에-스토크스 방정식을 푸는 방법입니다. 레이놀즈 응력을 모델링하여 방정식을 닫습니다. 계산 비용이 낮아 산업적으로 가장 널리 사용됩니다.

  • k-ε 모델 (Standard, RNG, Realizable): 난류 운동 에너지(k)와 난류 에너지 소산율(ε)에 대한 두 개의 추가 수송 방정식을 사용합니다. 범용성이 높지만 복잡 유동 예측에 한계가 있습니다.
  • k-ω 모델 (Standard, SST): 난류 운동 에너지(k)와 비타율(ω)에 대한 두 개의 수송 방정식을 사용합니다. SST k-ω 모델은 벽면 근처 및 유동 박리 예측에 우수합니다.
  • Spalart-Allmaras 모델 (SA): 단일 방정식 모델로, 주로 항공우주 분야 외부 유동에 사용됩니다.

LES (Large Eddy Simulation)

큰 규모의 와류는 직접 계산하고, 격자 해상도보다 작은 규모의 와류(SGS)는 모델링하는 방식입니다. RANS보다 정확하지만 계산 비용이 높습니다. 시간적/공간적으로 변동하는 난류 구조 포착에 적합합니다.

DNS (Direct Numerical Simulation)

나비에-스토크스 방정식을 모델링 없이 모든 시간 및 공간 스케일에 대해 직접 수치 해석합니다. 가장 정확하지만, 현실적인 공학 문제에는 계산 비용이 매우 높아 주로 기초 연구나 모델 검증에 사용됩니다.

난류 모델 선택 기준 및 한계

모델 계산 비용 정확도 (복잡 유동) 주요 장점 주요 단점 대표 적용 분야
RANS (k-ε) 낮음 보통 범용성, 계산 효율, 안정성 박리, 역압력 구배, 회전 유동 예측 부정확 일반 산업 유동, 내부 유동
RANS (k-ω SST) 낮음 우수 벽면 근처, 박리 예측 우수 일부 유동에서 과예측 가능 복잡 벽면 유동, 박리/재부착
RANS (SA) 낮음 보통 낮은 계산 비용, 특정 외부 유동 효율적 복잡 난류 구조 예측 한계 항공우주 외부 유동
LES 높음 매우 좋음 비정상/복잡 난류 구조 포착 높은 계산 비용, 벽면 근처 격자 요구 과다 제트, 와류 방출, 공력 소음
DNS 매우 높음 최고 난류의 모든 스케일 직접 해석 현재 공학 규모 적용 불가 난류 기초 연구, 모델 검증

어떤 난류 모델도 모든 유동에 대해 완벽한 예측 성능을 보장하지 않으므로, 해석 결과의 신뢰성 확보를 위해서는 항상 실험 데이터나 검증된 참고 문헌 결과와의 비교 및 검증 과정이 필수적입니다.
