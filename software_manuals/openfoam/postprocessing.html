OpenFOAM 사용 설명서: 후처리

OpenFOAM 해석이 완료되면, 생성된 데이터를 분석하고 시각화하는 후처리(Post-processing) 과정을 거칩니다. OpenFOAM은 자체적으로 몇 가지 후처리 유틸리티를 제공하며, 주로 ParaView와 같은 외부 오픈소스 시각화 도구를 많이 사용합니다.

1. ParaView 활용

ParaView는 OpenFOAM 결과를 시각화하는 데 가장 널리 사용되는 강력한 오픈소스 후처리 도구입니다. OpenFOAM 사례 디렉토리에서 다음 명령어를 통해 ParaView를 실행하고 결과를 불러올 수 있습니다:

paraFoam

또는 ParaView를 직접 실행한 후, OpenFOAM 사례 디렉토리 내에 빈 파일(예: case.foam 또는 case.OpenFOAM)을 생성하고 이 파일을 ParaView에서 열어 데이터를 불러올 수 있습니다. (touch case.foam)

ParaView 주요 기능

  • 데이터 불러오기 및 선택: 시간 스텝별 데이터, 변수(U, p, T, k 등) 선택.
  • 필터(Filters) 적용:
    • Slice: 특정 평면으로 단면을 잘라 변수 분포 확인.
    • Clip: 특정 조건(예: 특정 값 이상/이하)으로 영역을 잘라내기.
    • Contour: 등고선 또는 등가면 생성.
    • Glyph: 벡터(예: 속도)를 화살표 등으로 시각화.
    • Stream Tracer: 유선, 유맥선, 유적선 생성.
    • Calculator: 기존 변수들을 조합하여 새로운 변수 계산.
    • Plot Over Line / Plot Data: 특정 선을 따라 또는 특정 데이터 집합에 대해 그래프 생성.
  • 시각화 속성 조절: 색상 맵, 투명도, 객체 표현 방식 등 조절.
  • 애니메이션 제작: 비정상 상태 해석 결과를 시간에 따라 동영상으로 저장.
  • 데이터 내보내기: 시각화된 이미지, 데이터(CSV 등) 내보내기.
  • Python 스크립팅: 반복적인 후처리 작업 자동화.
(이미지: ParaView 인터페이스에서 OpenFOAM 결과(예: Cavity 유동)를 시각화하는 모습)

2. OpenFOAM 자체 후처리 유틸리티

OpenFOAM은 controlDict의 functions 항목이나 별도의 유틸리티를 통해 다양한 후처리 작업을 수행할 수 있습니다. 이러한 작업은 주로 계산 중 또는 계산 완료 후 실행되어 텍스트 파일 형태의 데이터를 생성합니다.

  • probes: 특정 위치에서 시간 변화에 따른 변수 값 추출.
  • sets / surfaces: 특정 셀 집합(set) 또는 면 집합(surface)을 정의하고, 이 영역에서 변수의 평균, 적분 값 등을 계산.
  • fieldAverage: 시간에 따른 필드 변수(속도, 압력 등)의 평균값 계산.
  • forceCoeffs / forces: 특정 경계면(patch)에 작용하는 힘(항력, 양력) 및 모멘트, 그리고 이들의 계수 계산.
  • wallShearStress: 벽면 전단 응력 계산.
  • yPlus: 벽면 근처 첫 번째 셀의 무차원 거리(y+) 계산.
  • sample: 특정 선, 면, 또는 체적에서 데이터를 샘플링하여 저장.
  • postProcess 유틸리티: -func 옵션과 함께 다양한 함수 객체(function objects)를 실행하여 후처리 데이터 생성. 
    postProcess -func "forceCoeffs(name=wing)" -latestTime

생성된 데이터는 주로 사례 디렉토리 내의 postProcessing 폴더에 저장되며, Gnuplot, Python (Matplotlib, Pandas), MATLAB 등 다른 도구를 사용하여 그래프로 시각화하거나 추가 분석할 수 있습니다.

3. 병렬 계산 결과 재구성

병렬 계산(mpirun 사용)을 수행한 경우, 결과는 각 프로세서별 디렉토리(processor0, processor1, ...)에 분할되어 저장됩니다. 이를 단일 시간 디렉토리로 합치기 위해서는 reconstructPar 유틸리티를 사용합니다.

reconstructPar -latestTime # 가장 최근 시간 스텝만 재구성
reconstructPar # 모든 시간 스텝 재구성

격자만 재구성하려면 reconstructParMesh를 사용합니다. 재구성된 데이터는 ParaView 등에서 불러와 후처리할 수 있습니다.

4. 데이터 변환

OpenFOAM 결과를 다른 후처리 소프트웨어나 파일 형식으로 변환해야 할 경우, 관련 유틸리티를 사용할 수 있습니다. (예: foamToVTK - VTK 형식으로 변환)

OpenFOAM의 후처리는 다양한 도구와 방법을 활용할 수 있으며, 해석 목적에 맞는 적절한 데이터 추출 및 시각화 방법을 선택하는 것이 중요합니다. ParaView는 강력한 시각화 기능을 제공하므로, 기본적인 사용법을 익혀두는 것이 매우 유용합니다.
