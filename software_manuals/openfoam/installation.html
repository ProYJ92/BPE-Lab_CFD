OpenFOAM 사용 설명서: 설치

OpenFOAM은 주로 Linux 환경에서 사용되지만, Windows 및 macOS에서도 다양한 방법으로 설치하여 활용할 수 있습니다. 이 섹션에서는 주요 운영체제별 설치 방법에 대해 간략히 안내합니다. OpenFOAM 버전(예: OpenFOAM Foundation 버전, ESI Group 버전) 및 배포판에 따라 세부 절차가 다를 수 있으므로, 공식 웹사이트의 최신 설치 가이드를 반드시 참조하십시오.

1. Linux 환경 설치

Linux는 OpenFOAM의 주요 개발 및 실행 환경입니다. Ubuntu, CentOS, Fedora 등 다양한 배포판을 지원합니다.

Ubuntu (Debian 기반)

  • 패키지 매니저(APT) 사용: OpenFOAM Foundation 또는 ESI Group에서 제공하는 공식 저장소를 추가한 후, apt-get 명령어를 통해 간편하게 설치할 수 있습니다. 
    sudo sh -c "wget http://dl.openfoam.org/gpg.key -O - | apt-key add -"
sudo add-apt-repository http://dl.openfoam.org/ubuntu
sudo apt-get update
sudo apt-get install openfoam8 # 또는 원하는 버전
  • 소스 코드 컴파일: 공식 웹사이트에서 소스 코드를 다운로드하여 직접 컴파일할 수 있습니다. 이 방법은 특정 라이브러리 의존성 관리 및 사용자 정의 컴파일 옵션 설정에 유리하지만, 시간이 오래 걸리고 복잡할 수 있습니다.

CentOS / RHEL / Fedora (RPM 기반)

  • 패키지 매니저(YUM/DNF) 사용: 유사하게 공식 저장소를 추가하고 패키지 매니저를 통해 설치할 수 있습니다.
  • Docker 컨테이너 활용: Docker Hub에 배포된 OpenFOAM 이미지를 사용하여 격리된 환경에서 실행할 수 있습니다.

2. Windows 환경 설치

Windows에서 OpenFOAM을 사용하는 방법은 주로 다음 세 가지입니다.

  • WSL (Windows Subsystem for Linux): Windows 10/11에서 제공하는 WSL을 설치하고, WSL 환경 내에서 Linux용 OpenFOAM을 설치하는 방법입니다. 가장 네이티브에 가까운 성능을 제공하며 권장되는 방식입니다. WSL2 사용 시 Docker Desktop과 연동하여 Docker 컨테이너를 활용할 수도 있습니다.
  • Docker Desktop for Windows: Docker Hub의 OpenFOAM 이미지를 다운로드하여 컨테이너 환경에서 실행합니다. GUI 애플리케이션(예: ParaView) 연동을 위해 X 서버 설정이 필요할 수 있습니다.
  • 네이티브 Windows 버전 (예: blueCFD, CFDSupport): 일부 커뮤니티 또는 기업에서 MinGW/MSYS2 등을 사용하여 Windows 환경에서 직접 컴파일된 OpenFOAM 버전을 배포하기도 합니다. 설치는 간편할 수 있으나, Linux 버전과의 호환성이나 최신 버전 지원이 다를 수 있습니다.
(이미지: WSL 환경에서 OpenFOAM 실행하는 터미널 화면 또는 Docker Desktop 인터페이스)

3. macOS 환경 설치

  • Docker Desktop for Mac: Windows와 유사하게 Docker Hub의 OpenFOAM 이미지를 사용하여 컨테이너 환경에서 실행하는 것이 일반적입니다.
  • 소스 코드 컴파일 (Homebrew 등 활용): Homebrew와 같은 패키지 매니저를 통해 필요한 라이브러리를 설치하고 소스 코드를 직접 컴파일할 수 있으나, 의존성 문제가 발생할 수 있습니다.

설치 후 환경 설정

OpenFOAM을 설치한 후에는 터미널에서 OpenFOAM 명령어와 라이브러리를 사용할 수 있도록 환경 변수를 설정해야 합니다. 일반적으로 설치된 OpenFOAM 디렉토리 내의 etc/bashrc 또는 etc/cshrc 파일을 .bashrc (또는 .zshrc 등 쉘 설정 파일)에 추가하여 설정합니다.

# 예시: .bashrc 파일에 다음 내용 추가
# source /opt/openfoam8/etc/bashrc  (설치 경로에 따라 다름)

설정이 완료되면 새 터미널을 열고 simpleFoam -help 와 같은 명령어를 실행하여 OpenFOAM이 정상적으로 인식되는지 확인합니다.
