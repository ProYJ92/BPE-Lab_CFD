Ansys Fluent 사용 설명서: 솔버 설정 및 실행

전처리 단계에서 물리 모델, 물성치, 경계 조건 등이 설정되면, 다음은 솔버(Solver) 관련 설정을 구성하고 해석을 실행하는 단계입니다. 이 단계에서는 수치 해석 기법, 이산화 차수, 수렴 기준 등을 정의하여 정확하고 안정적인 해를 얻는 것을 목표로 합니다.

1. 수치 해석 방법 (Solution Methods)

Fluent는 압력-속도 연계(Pressure-Velocity Coupling) 방식, 공간 이산화(Spatial Discretization) 기법 등을 사용자가 선택할 수 있도록 합니다.

  • Pressure-Velocity Coupling Scheme:
    • SIMPLE, SIMPLEC, PISO: 주로 Pressure-Based Solver에서 사용되는 전통적인 분리형(segregated) 알고리즘입니다. (각 알고리즘 특징은 CFD 이론: 솔버 섹션 참조)
    • Coupled: 압력과 운동량 방정식을 동시에 푸는 연립형(coupled) 알고리즘으로, 수렴성이 우수하지만 반복 계산 당 비용이 높을 수 있습니다. 고속 압축성 유동이나 복잡한 물리 현상에 유리합니다.
  • Spatial Discretization (공간 이산화): 각 변수(운동량, 난류 변수 등)에 대한 대류항(convection term)과 확산항(diffusion term)의 이산화 차수를 설정합니다.
    • Gradient: Least Squares Cell Based (기본값, 대부분의 경우 적합), Green-Gauss Cell Based, Green-Gauss Node Based 등.
    • Pressure: Standard, Second Order, PRESTO!, Body Force Weighted 등. 압력장의 특성에 따라 선택합니다.
    • Momentum, Turbulent Kinetic Energy, etc.: First Order Upwind (안정적이나 수치 확산 큼, 초기 계산에 사용), Second Order Upwind (정확도 높음, 권장), QUICK 등. 일반적으로 Second Order Upwind를 사용합니다.
(이미지: Fluent의 Solution Methods 설정 창)

2. 제어 (Solution Controls)

해석의 안정성과 수렴 속도를 조절하기 위한 파라미터를 설정합니다.

  • Under-Relaxation Factors (감속 계수): 반복 계산 시 변수 값의 변화량을 조절하여 해의 발산을 방지하고 안정적인 수렴을 돕습니다. 기본값을 사용하거나, 수렴이 불안정할 경우 값을 낮춰볼 수 있습니다. (Coupled 알고리즘에서는 일반적으로 필요 없음)
  • Limits: 계산 중 발생할 수 있는 비현실적인 값(예: 음수 온도)을 제한하여 솔버의 안정성을 높입니다.

3. 모니터링 (Monitors)

해석 진행 상황과 수렴 여부를 판단하기 위해 잔차(Residuals) 및 특정 물리량(예: 항력, 양력, 특정 지점의 속도/압력)을 모니터링합니다.

  • Residuals: 각 지배 방정식의 잔차 그래프를 확인합니다. 일반적으로 잔차가 10-3 ~ 10-6 이하로 감소하고 안정화되면 수렴된 것으로 판단합니다 (문제의 특성에 따라 기준 다름).
  • Report Definitions / Surface Monitors: 관심 있는 물리량(예: 특정 면에서의 평균 압력, 물체 표면의 항력 계수)을 정의하여 반복 계산에 따른 변화를 그래프로 확인합니다. 이 값들이 일정한 값으로 안정화되는 것도 수렴 판단의 중요한 기준입니다.
(이미지: Fluent의 잔차 모니터 및 관심 물리량 모니터 그래프)

4. 초기화 (Solution Initialization)

반복 계산을 시작하기 전, 유동장 전체의 초기값을 설정합니다. 적절한 초기화는 수렴 속도를 높이고 안정적인 해를 얻는 데 도움이 됩니다.

  • Hybrid Initialization: Fluent가 자동으로 유동장을 초기화하는 방법으로, 대부분의 경우 효과적입니다.
  • Standard Initialization: 사용자가 직접 모든 변수의 초기값을 입력하거나, 특정 경계면(예: 입구)의 값으로 전체 영역을 초기화합니다.
  • Patch: 특정 영역(Cell Zone)에 대해 국소적으로 다른 초기값을 설정할 수 있습니다.

5. 계산 실행 (Run Calculation)

모든 설정이 완료되면 계산을 실행합니다.

  • Number of Iterations (정상 상태 해석): 최대 반복 계산 횟수를 지정합니다. 수렴이 완료되면 지정된 횟수 이전에 계산이 종료될 수 있습니다.
  • Time Step Size & Number of Time Steps (비정상 상태 해석): 시간 간격(Δt)과 전체 계산 시간 스텝 수를 지정합니다. CFL 조건 등을 고려하여 적절한 시간 간격을 선택해야 합니다.
  • Data File Quantities (자동 저장): 비정상 상태 해석 시 특정 시간 간격마다 또는 정상 상태 해석 시 특정 반복 횟수마다 중간 결과 파일(.dat)을 자동으로 저장하도록 설정할 수 있습니다.
  • Calculation Activities: 계산 중 특정 작업(예: Solution Animations, User-Defined Function 실행)을 수행하도록 설정할 수 있습니다.

계산이 진행되는 동안 잔차 및 모니터링하는 물리량의 변화를 주시하며 수렴 여부를 판단합니다. 필요시 계산을 중지하고 설정을 조정한 후 다시 시작할 수 있습니다.
